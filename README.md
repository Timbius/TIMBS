1.Тема и концепция

Название - 
 — это современное веб-приложение для онлайн-записи в барбершоп. Платформа позволяет клиентам выбрать услугу, мастера и удобное время посещения без необходимости звонка администратору. Пользователи могут управлять своими записями, и личными данными через персональный кабинет, просматривать каталог услуг, их описание и стоимость, самостоятельно выбрать мастера и удобное время для записи.  Приложение упрощает процесс бронирования и записи делая взаимодействие с барбершопом быстрым и удобным.

Аудитория:
Приложение созданно для мужчин любых возрастов, которые знают толк в уходе за собой и регулярно посещают барбершоп, для клиентов, которые предпочитают онлайн-запись вместо телефонных звонков или постоянных очередей, для людей, которые ценят своё время и комфорт


Ключевой функционал:

Аутентификация и регистрация пользователей
Создание аккаунта (email/телефон + пароль).
Авторизация в системе.
Восстановление пароля.

Выбор услуг и мастеров
Просмотр списка доступных услуг (стрижка, бритьё, укладка и др.).
Отображение стоимости и продолжительности услуги.
Выбор конкретного мастера с просмотром его профиля и рейтинга.

Онлайн-запись на услугу
Выбор удобной даты и времени из доступных слотов.
Подтверждение записи.
Просмотр и отмена предстоящих записей.

Личный кабинет пользователя
Редактирование персональных данных.
Просмотр истории посещений.
Управление активными записями.

2.Структура приложения
1) Главная (/) - основная страница сайта
2) Каталог услуг (/catalog) - список услуг
3) Подробная страница услуги (/catago/:id) - подробное описание услуги и цена
4) Страница мастеров (/barbers) - список мастеров
5) Подробная страница мастера (/barbers/:id) - подробная информация про мастера
6) Страница онлайн-записи (/records) - страница для записи на услугу
7) Поиск (/search) - поиск по услуге или по мастеру
8) Аутентификация (/auth/*) -регистация / вход / восстановление пароля
9) Личный кабинет (/profile) - кабинет с информацией пользователя 
10) Админ панель (/admin) - Раздел доступный только пользователям с ролью администратор, позволяющая управлять учетными записями

3. Технологический стек
frontend: Vanilla JavaScript
Без фреймворков
 
 Причины выбора:
 Демонстрация базовых навыков разработки
 Контоль архитектуры 

backend
Node.js
Express.js

Причины выбора Node.js:
Использование JavaScript и на frontend, и на backend — единая экосистема разработки.
Высокая производительность при работе с асинхронными запросами (поиск, авторизация, работа с БД).
REST API.

Причины выбора Express.js:
Лёгкий и минималистичный фреймворк для Node.js.
Удобная маршрутизация.
Поддержка middleware.
Простота REST API.

База данных
MySQL
 Причины выбора:
 Реаляционная структура данных
 Поддержка связей
 Стабильность
 Производительность

 Инфраструктура
 Frontend - static hosting
 Backend - Node.js Server
 Database - MySQL
